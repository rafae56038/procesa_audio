Guía de instalación

procesa_audio — análisis de sentimiento en audio con IA

Requisitos previos Opción 1 — CPU Opción 2 — NVIDIA Opción 3 — AMD ROCm Resumen

Para todos los casos

Requisitos previos

  • Python 3.10 o superior
  • git instalado
  • Al menos 6 GB de RAM libre
git clone <url-del-repo>
cd procesa_audio
python -m venv venv

Opción 1

CPU

Sin GPU — Windows o Linux

~50-90 seg por audio

Funciona en cualquier máquina. No requiere configuración especial.

1Activar entorno virtual
# Windows
venv\Scripts\activate

# Linux
source venv/bin/activate
2Instalar dependencias
pip install torch torchvision torchaudio
pip install faster-whisper transformers librosa pysentimiento

Scripts disponibles: procesar_audio_vf.py, procesar_audio_chunk.py

Opción 2

NVIDIA

GPU NVIDIA + CUDA — Windows o Linux

~5-10 seg por audio

Requiere GPU NVIDIA con al menos 6 GB VRAM. Recomendado: RTX 3060 12 GB o superior.

1Verificar que CUDA está instalado
nvidia-smi

Debes ver el driver y la versión de CUDA. Si no aparece, instala los drivers desde el sitio de NVIDIA.

2Activar entorno virtual
# Windows
venv\Scripts\activate

# Linux
source venv/bin/activate
3Instalar PyTorch con CUDA

Elige según la versión de CUDA que reportó nvidia-smi:

# CUDA 12.1 (más común en drivers recientes)
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121

# CUDA 11.8
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
4Resto de dependencias
pip install faster-whisper transformers librosa pysentimiento
5Verificar
python -c "import torch; print(torch.cuda.is_available()); print(torch.cuda.get_device_name(0))"

Debe imprimir True y el nombre de tu GPU.

Scripts disponibles: todos, incluyendo test_gpu.py

Opción 3

AMD

GPU AMD + ROCm — solo Linux

~8-15 seg por audio

Requiere GPU AMD RDNA2 o superior (RX 6000 series en adelante). La RX 6650 XT es compatible (gfx1032).

Windows con AMD no está soportado por estas librerías. Usar Linux (Ubuntu 22.04 recomendado).
1Instalar ROCm

Para Ubuntu 22.04:

sudo apt update
wget https://repo.radeon.com/amdgpu-install/6.0/ubuntu/jammy/amdgpu-install_6.0.60000-1_all.deb
sudo dpkg -i amdgpu-install_6.0.60000-1_all.deb
sudo amdgpu-install --usecase=rocm
sudo usermod -aG render,video $LOGNAME
# Reiniciar sesión
2Verificar ROCm
rocm-smi

Debes ver tu GPU listada.

3Activar entorno virtual e instalar PyTorch con ROCm
source venv/bin/activate
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/rocm6.0
4Resto de dependencias
pip install transformers librosa pysentimiento
faster-whisper usa CTranslate2 que no soporta ROCm. El script test_gpu.py usa Whisper vía transformers, que sí es compatible.
5Verificar
python -c "import torch; print(torch.cuda.is_available()); print(torch.cuda.get_device_name(0))"

En ROCm, PyTorch expone la GPU AMD bajo el nombre cuda. Debe imprimir True.

Scripts disponibles: test_gpu.py
